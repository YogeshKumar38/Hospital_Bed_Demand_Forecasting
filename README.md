# Hospital_Bed_Demand_Forecasting
Hospital Bed Demand Foecasting project with a streamlit dashboard to visualize trends, analyzedatasets, and predict future bed requirements efficiently
A machine learning powered Streamlit dashboard that predicts hospital bed demand using historical hospital data.

## Features

- Machine Learning prediction using Random Forest
- Interactive Streamlit dashboard
- Bed utilisation analysis
- Patient refusal rate analysis
- Staff attendance analysis
- Demand clustering using KMeans

## Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-Learn
- Plotly

## Installation

Clone the repository

git clone https://github.com/YOUR_USERNAME/hospital-bed-demand-dashboard.git

Install dependencies
pip install -r requirements.txt

Run the dashboard

streamlit run app.py
